JavaScript for beginners

Здесь вы можете изучить JavaScript с нуля. JavaScript - это мультипарадигменный язык программирования, поддерживающий ООП. Обычно используется в браузерах, для добавления интерактивности веб-страницам, то есть позволяет применять сложные вещи на веб-странице тогда, когда на ней происходит что-то еще, кроме статичного отображения:

  • Периодически обновляемый контент
  • 2D, 3D графика, анимация
  • Видеопроигрыватель

    • HTML - первый слой, язык разметки нашей страницы, CSS - второй, добавляет нашей странице стиль и наконец третий слой, это JavaScript. Создает динамически обновляемый контент, мультимедиа вашей страницы
